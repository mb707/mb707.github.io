Responsible Use

AI agents can make mistakes. They may generate code that looks correct but has bugs, security issues, or missing details. Because of this, programmers should test AI-generated code before using it in a real project.

Developers also need to think about privacy, copyright, bias, and academic honesty. AI can be a useful learning tool, but students should understand and be able to explain the work they submit.
