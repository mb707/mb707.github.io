CST 336 HW 3 - Pokémon Research Terminal

FILES
- index.html
- css/style.css
- js/script.js

HOW TO RUN
Open index.html in a browser with internet access.
For the most reliable results can ALSO use Live Server in Visual Studio Code.

FEATURES
- Uses fetch() with async/await
- Accepts Pokémon name or National Pokédex number
- Random Pokémon button
- Retrieves Pokémon and species data from PokéAPI
- Displays official artwork, types, height, weight, abilities, stats, moves, habitat, generation, and Pokédex text
- Changes the color theme based on primary type
- Includes loading and error states
- Stores the five most recent successful searches in localStorage
- Uses sessionStorage to avoid repeating identical API requests during a session

API
https://pokeapi.co/api/v2/
No API key is required.

FINAL SUBMISSION CHECK
Run index.html through the W3C HTML Validator and style.css through the W3C CSS Validator before submitting.
